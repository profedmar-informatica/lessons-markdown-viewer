# Bem-vindo à sua Escrivaninha Digital!

Este é um visualizador de arquivos Markdown com um toque vintage.

## Como funciona?

No menu lateral à esquerda, você encontrará uma lista de documentos. Clique em qualquer um deles para carregar seu conteúdo aqui, como se estivesse folheando papéis em uma escrivaninha antiga.

### Recursos:

*   **Navegação Simples:** Clique nos títulos para ver o conteúdo.
*   **Estilo Agradável:** Desfrute de uma experiência de leitura que simula documentos impressos.
*   **Conteúdo Dinâmico:** Todos os arquivos Markdown são carregados sob demanda.

Esperamos que você goste da sua nova escrivaninha!
