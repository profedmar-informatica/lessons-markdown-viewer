# Como Usar a Escrivaninha Digital

É muito simples navegar e interagir com este visualizador de documentos.

## 1. Menu Lateral

À esquerda, você verá uma lista de botões. Cada botão corresponde a um arquivo Markdown (`.md`) que está armazenado na pasta `/content` do servidor.

## 2. Selecionando um Documento

*   **Clique** em qualquer um dos botões no menu lateral.
*   O conteúdo do arquivo Markdown correspondente será **carregado dinamicamente** e exibido na área principal, à direita.

## 3. Lendo o Conteúdo

*   O conteúdo é formatado para parecer uma folha de papel, com uma tipografia que simula um documento impresso.
*   Você pode rolar para baixo para ler documentos mais longos. A rolagem foi projetada para manter a sensação de folhas empilhadas.

## 4. Adicionando Novos Documentos

Para adicionar seus próprios documentos:

1.  Crie um novo arquivo com a extensão `.md` (ex: `meu-documento.md`).
2.  Salve este arquivo dentro da pasta `/content` do projeto.
3.  Atualize a página no navegador. O novo documento aparecerá automaticamente no menu lateral!

Aproveite a sua leitura!
