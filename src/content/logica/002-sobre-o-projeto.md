# Sobre Este Projeto

Este projeto foi desenvolvido como uma demonstração de uma aplicação full-stack simples, combinando um backend Node.js/Express com um frontend de HTML, CSS e JavaScript puro.

## Tecnologias Utilizadas

*   **Backend:** Node.js com Express para servir APIs e arquivos estáticos.
*   **Frontend:** HTML5, CSS3 e JavaScript ES6+.
*   **Renderização Markdown:** A biblioteca `marked` é usada no frontend para converter o Markdown em HTML.

## Design e Estética

O objetivo principal do design foi criar uma experiência imersiva que remetesse a uma escrivaninha antiga. Isso foi alcançado através de:

*   **Textura de Madeira:** Um plano de fundo que simula uma escrivaninha de madeira escura.
*   **Folhas de Papel:** O conteúdo é exibido em "folhas de papel" com fundo amarelado, sombras suaves e bordas discretas.
*   **Tipografia:** Fontes que lembram documentos impressos para uma leitura mais agradável.
*   **Empilhamento:** A ilusão de folhas sobrepostas para dar profundidade.

Este projeto é um exemplo de como a funcionalidade pode ser combinada com uma estética cuidadosa para criar uma experiência de usuário única.
